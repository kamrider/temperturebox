# STM32温度监控系统

基于STM32的实时温度监控系统，使用DS18B20温度传感器采集数据，通过OLED显示实时温度。

## 硬件需求

- STM32开发板
- DS18B20温度传感器
- OLED显示屏

## 功能特点

- 实时温度采集
- OLED显示温度数据
- 温度精确到小数点后一位
- 500ms刷新一次数据

## 接线说明

- DS18B20: 接入PGx端口
- OLED: IIC接口，接入SCL/SDA

## 使用说明

1. 按照接线说明连接硬件
2. 编译并下载程序
3. 系统将自动开始温度监控并显示

## 开发环境

- Keil MDK
- STM32标准库
